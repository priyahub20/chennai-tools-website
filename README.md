 About the Project

This website is developed for **Chennai Tools**, a company specializing in high-quality spare parts for paper cup making machines.

The goal of this website is to present the company in a modern, professional, and user-friendly way, making it easier for customers to explore products and contact the business.

 What We Provide

* Machine Components
* Heating Elements
* Bearings & Mechanical Parts
* Sensors & Electrical Parts
* Custom Spare Parts


 Key Features

* Clean and modern UI design
* Fully responsive layout
* Interactive hover effects
* Dedicated pages for Home, About, Spare Parts, and Contact
